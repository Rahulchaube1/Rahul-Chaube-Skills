# Learning Pathways Overview

Welcome to the **RCS Learning Pathways**. These guides are designed to take you from a developer writing your first API request to an expert optimizing large-scale serving clusters and distributed training graphs.

---

## 🗺️ Choose Your Path

Select a pathway based on your experience and career objectives:

| Pathway                                                           | Target Audience                               | Key Concepts                                         |
| ----------------------------------------------------------------- | --------------------------------------------- | ---------------------------------------------------- |
| [Beginner AI Engineer](beginner-ai-engineer.md)                   | Software developers new to AI integration.    | Prompt variables, APIs, JSON Outputs, local models.  |
| [Intermediate Agent Developer](intermediate-agentic-developer.md) | Developers building interactive flows.        | ReAct loops, planning, tools integrations, state.    |
| [Advanced Cognitive Architect](advanced-cognitive-architect.md)   | Systems engineers designing autonomous nodes. | GraphRAG, Tree of Thoughts, context engineering.     |
| [Expert Systems Optimizer](expert-systems-optimization.md)        | Infrastructure architects scaling models.     | Serving (vLLM), KV cache, quantization, distributed. |
