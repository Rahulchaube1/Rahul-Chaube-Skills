# AI Engineer Curriculums

This document outlines structural curricula to help you master AI engineering across different timelines.

---

## 30-Day Learning Plan: Agent Fundamentals

- **Goal**: Build and deploy a single-agent terminal developer tool with tool integrations.
- **Timeline**:
  - **Days 1–7**: Fundamentals of structured prompts and JSON outputs.
  - **Days 8–14**: Tool calling and function schemas.
  - **Days 15–22**: Building a basic ReAct loop in Python.
  - **Days 23–30**: Final Project - Coding assistant that formats and runs lint tests on git changes.

---

## 90-Day Mastery Roadmap: Production Cognitive Systems

- **Goal**: Build and evaluate multi-agent reasoning graphs with GraphRAG.
- **Timeline**:
  - **Days 1–30**: Master Agent frameworks (LangGraph, CrewAI). Build planning, routing, and memory loops.
  - **Days 31–60**: Knowledge Retrieval and Hybrid Search (Neo4j, Vector Databases, Chunking).
  - **Days 61–90**: Final Project - Create an Enterprise Knowledge base that answers queries by traversing knowledge graphs and vectors.

---

## 365-Day AI Engineer Curriculum: Systems & Research

- **Goal**: Optimize model serve latencies, fine-tune adapters, and compile custom pipelines.
- **Timeline**:
  - **Months 1–3**: Deepening agent architectures, memory compactions, and advanced evaluations.
  - **Months 4–6**: Local deployment (vLLM, Ollama), GPU memory metrics, and PagedAttention.
  - **Months 7–9**: LoRA, QLoRA fine-tuning, synthetic data generation, and DPO alignments.
  - **Months 10–12**: Distributed setups (DeepSpeed, Megatron), CUDA optimization, and hosting serving nodes.
