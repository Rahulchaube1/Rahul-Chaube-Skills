# Reference Project Blueprints

This directory contains full architectural and implementation blueprints for five production-grade AI agents and systems.

---

## 🚀 Reference Catalog

| Project                                       | Description                        | Architecture Highlights                                       |
| --------------------------------------------- | ---------------------------------- | ------------------------------------------------------------- |
| [Coding Agent](coding-agent.md)               | Autonomous local coder.            | Linter feedback loops, git stage checks, AST verification.    |
| [Deep Research Agent](deep-research-agent.md) | Literature review synthesizer.     | Parallel web searches, source verification, citation tagging. |
| [Browser Agent](browser-automation-agent.md)  | Vision-guided computer operator.   | Page screenshots, coordinate grids, DOM mapping.              |
| [GraphRAG Assistant](graph-rag-assistant.md)  | Neo4j + Vector search engine.      | Hybrid routing, entity mapping, contextual merge.             |
| [Voice Agent](voice-agent.md)                 | Ultra-low latency voice companion. | STT / TTS streams, sentence buffering, interrupt handling.    |

---

## 🛠️ Design Standards

Every project blueprint implements:

- **Architectural Diagrams**: Mermaid blocks detailing communication flows.
- **Folder Structures**: Clear organization of files.
- **Best Practices**: Performance tuning and cost bounds.
- **Security Checklists**: Secrets, inputs validation, and sandboxing rules.
