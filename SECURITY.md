# Security Policy

## Supported Versions

Only the latest major version of **Rahul-Chaube-Skills (RCS)** is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, such as unsafe prompt injection guidelines, information leaks in guidelines, or malicious scripts in templates, please report it immediately.

Do NOT open a public issue. Email us directly at `security@rahulchaube.com`.

We will acknowledge your report within 48 hours and provide a detailed timeline for investigation, patching, and public disclosure.
