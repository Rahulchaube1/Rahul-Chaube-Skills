# Image Generation

Prompts formatting, style weights, aspect ratios, and seed settings.

## Core Guidelines

- **Style Weights**: Specify texture, lighting, and style parameters clearly.
- **Parameters**: Pass correct aspect ratio parameters (e.g. --ar 16:9).
- **Seeds**: Keep seeds consistent when performing style variations.
