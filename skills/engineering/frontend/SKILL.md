# Frontend

UI building, CSS styling, components structure, and layouts.

## Core Guidelines

- **Responsive Layouts**: Design for both desktop and mobile viewports.
- **Style Consistency**: Utilize global CSS variables or design system tokens. Avoid ad-hoc styling.
- **Semantic HTML**: Use proper HTML5 elements (`main`, `article`, `section`, `nav`).
