# Cloud

Cloud services, IAC setups, serverless patterns, and configuration.

## Core Guidelines

- **Infrastructure as Code**: Manage cloud resources using Terraform, CloudFormation, or Pulumi.
- **Security Grouping**: Follow the principle of least privilege for IAM permissions and firewall rules.
- **Monitoring**: Set up logging and alerts for database/compute health.
