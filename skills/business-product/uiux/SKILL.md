# Uiux

User interface design, Figma flows, design patterns, and layouts.

## Core Guidelines

- **Design Tokens**: Standardize colors, font weights, and spacing scales.
- **Accessibility (a11y)**: Adhere to WCAG standards (color contrasts, screen reader labels).
- **Usability**: Minimize user steps to complete key tasks.
