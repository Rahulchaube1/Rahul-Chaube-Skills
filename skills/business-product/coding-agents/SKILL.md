# Coding Agents

Guidelines for orchestrating multiple coding agents in teams.

## Core Guidelines

- **Task Separation**: Divide big tasks into small, isolated agent assignments.
- **State Coordination**: Synchronize agent context folders using structured JSON schemas.
- **Verification Nodes**: Install dedicated evaluator nodes to check code before final output.
