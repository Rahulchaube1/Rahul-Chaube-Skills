# Communication

Collaborative communication, active listening, and alignment protocols.

## Core Guidelines

- **Explicit Alignment**: Highlight the specific trade-offs of proposed plans clearly (e.g. speed vs safety).
- **Proactive Warnings**: Inform the user of breaking changes before committing code.
- **Concise Summaries**: Conclude task execution turns with clean walkthrough checklists.
