# Productivity

Optimizing execution speed, keyboard shortcuts, and agent velocity.

## Core Guidelines

- **Shortcut Actions**: Use CLI tools and shortcuts where possible instead of long manual actions.
- **Incremental Steps**: Commit changes incrementally to avoid large merge conflicts or massive rollback steps.
- **Clean Environment**: Keep scratch directories and temp files organized.
