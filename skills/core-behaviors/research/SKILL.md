# Research

Codebase exploration and symbol tracing methodologies.

## Core Guidelines

- **Explore**: List directory hierarchies before viewing specific code details.
- **Grep**: Search for symbol usages globally to map potential collateral damage.
- **Read Configuration**: Review dependency manifests (`package.json`, `requirements.txt`) first to understand package bounds.
