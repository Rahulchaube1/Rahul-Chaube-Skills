# Writing

Technical writing, grammar, documentation, and formatting standards.

## Core Guidelines

- **Clarity**: Use active, concise language. Avoid passive phrasing and filler words.
- **Formatting**: Maintain clean markdown headers, bullets, and tables. Avoid deeply nested structures.
- **Tone**: Keep communication professional, helpful, and action-oriented.
